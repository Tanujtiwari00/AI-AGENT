# 🚀 AI Career Agent Pro

An advanced AI-powered career recommendation web application built using Python and Streamlit. The app analyzes user skills, ranks multiple job roles using match scores, and provides personalized upskilling guidance.

## 🔥 Features

- Analyze user skills
- Recommend top matching careers
- Match score for multiple job roles
- Missing skills detection
- Personalized learning roadmap
- Resume guidance
- Interactive dashboard with charts

## 💼 Roles Covered

- Data Analyst
- Business Analyst
- Python Developer
- Frontend Developer
- Backend Developer
- AI Engineer
- ML Engineer
- BI Analyst
- MIS Executive
- Product Analyst

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
